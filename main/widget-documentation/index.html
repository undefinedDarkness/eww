    Widgets

    widget

    these properties apply to all widgets, and can be used anywhere!

    Properties

    • class: css class name
    • valign: how to align this vertically. possible values: "fill", "baseline", "center", "start", "end"
    • halign: how to align this horizontally. possible values: "fill", "baseline", "center", "start", "end"
    • width: width of this element. note that this can not restrict the size if the contents stretch it
    • height: height of this element. note that this can not restrict the size if the contents stretch it
    • active: If this widget can be interacted with
    • tooltip: tooltip text (on hover)
    • visible: visibility of the widget
    • style: inline css style applied to the widget
    • onscroll: event to execute when the user scrolls with the mouse over the widget
    • onhover: event to execute when the user hovers over the widget

    combo-box-text

    A combo box allowing the user to choose between several items.

    Properties

    • items: Items that should be displayed in the combo box
    • onchange: runs the code when a item was selected, replacing {} with the item as a string

    expander

    A widget that can expand and collapse, showing/hiding it's children.

    Properties

    • name: name of the expander
    • expanded: sets if the tree is expanded
    • vexpand: should this container expand vertically
    • hexpand: should this container expand horizontally

    a checkbox

    A checkbox that can trigger events on checked / unchecked.

    Properties

    • onchecked: action (command) to be executed when checked by the user
    • onunchecked: similar to onchecked but when the widget is unchecked

    color-button

    A button opening a color chooser window

    Properties

    • use-alpha: bool to whether or not use alpha
    • onchange: runs the code when the color was selected

    color-chooser

    A color chooser widget

    Properties

    • use-alpha: bool to wether or not use alpha
    • onchange: runs the code when the color was selected

    scale

    A slider.

    Properties

    • flipped: flip the direction
    • draw-value: draw the value of the property
    • value: the value
    • min: the minimum value
    • max: the maximum value
    • onchange: command executed once the value is changes. The placeholder {}, used in the command will be replaced by the new value.

    progress

    A progress bar

    Properties

    • flipped: flip the direction
    • value: value of the progress bar (between 0-100)
    • orientation: orientation of the progress bar. possible values: "vertical", "v", "horizontal", "h"

    input

    An input field. For this to be useful, set focusable="true" on the window.

    Properties

    • value: the content of the text field
    • onchange: Command to run when the text changes. The placeholder {} will be replaced by the value

    button

    A button

    Properties

    • onclick: a command that get's run when the button is clicked
    • vexpand: should this container expand vertically
    • hexpand: should this container expand horizontally

    image

    A widget displaying an image

    Properties

    • path: path to the image file
    • width: width of the image
    • height: height of the image

    box

    the main layout container

    Properties

    • spacing: spacing between elements
    • orientation: orientation of the box. possible values: "vertical", "v", "horizontal", "h"
    • space-evenly: space the widgets evenly.
    • vexpand: should this container expand vertically
    • hexpand: should this container expand horizontally

    label

    A text widget giving you more control over how the text is displayed

    Properties

    • text: the text to display
    • limit-width: maximum count of characters to display
    • markup: Pango markup to display
    • wrap: Wrap the text. This mainly makes sense if you set the width of this widget.

    literal

    A widget that allows you to render arbitrary XML.

    Properties

    • content: inline Eww XML that will be rendered as a widget.

    calendar

    A widget that displays a calendar

    Properties

    • day: the selected day
    • month: the selected month
    • year: the selected year
    • show-details: show details
    • show-heading: show heading line
    • show-day-names: show names of days
    • show-week-numbers: show week numbers
    • onclick: command to run when the user selects a date. The {} placeholder will be replaced by the selected date.
